<p align="center">
    基于 <a href="https://facebook.github.io/react">React</a> 和 <a href="https://koa.bootcss.com/">Koa</a> 的完整前端技术解决方案。

<br />
<strong>易于开发</strong>：一套代码输出多端；零配置支持热更新、TS
<br />
<strong>功能丰富</strong>：多语言；按需载入；PWA；CSS 命名空间
</p>

## 快速使用

1. 初始化项目

```bash
> npx koot-cli
```

2. 运行开发环境

```bash
> cd [项目名] && npm run dev
```
