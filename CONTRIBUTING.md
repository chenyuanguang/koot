# 向 Koot.js 开源项目提出贡献

👍🎉 感谢您百忙之中抽出时间为 _Koot.js_ 提出贡献 🎉👍

您可以选择如下方式为 _Koot.js_ 添砖加瓦 🧱

-   提出意见、建议
-   提交 Bug 报告
-   编写代码并发起 Pull Request

---

## 提出意见、建议

如果您有任何意见和建议，欢迎提交到 [Issue 列表](https://github.com/cmux/koot/issues)

## 提交 Bug 报告

如果您遇到过任何 Bug 或不可解释的问题，请将提交到 [Issue 列表](https://github.com/cmux/koot/issues)，在提交 Bug 报告时，建议附带以下信息

-   _Koot.js_ 版本
-   项目类型 (SSR / SPA)
-   出现问题的环境 (客户端 / 服务器端 / 生产环境 / 开发环境)
-   问题可否复现

## 编写代码并发起 Pull Request

如果您有意愿，也可以直接对 _Koot.js_ 的代码进行修改，并提交。请依照以下步骤进行开发：

1. Fork [_Koot.js_ 代码仓库](https://github.com/cmux/koot/)
2. 在本地代码目录内运行 `npm run bootstrap`
3. _进行开发..._
4. 编写测试
5. 提交 [Pull Request](https://github.com/cmux/koot/pulls)
