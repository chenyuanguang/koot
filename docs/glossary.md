# 技术术语

| 术语         | 解释                                                                                                                          |
| ------------ | ----------------------------------------------------------------------------------------------------------------------------- |
| `SSR`        | Server-Side Rendering，服务端渲染，本文档泛指 React 在 Node 服务端渲染。                                                      |
| `Isomorphic` | 同构模式，指同一套代码在服务端和客户端都可以构建，本文档泛指一套 React 客户端代码可以在浏览器和服务端共用，达到渲染效果一致。 |
