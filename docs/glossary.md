

# 相关

### 术语说明


术语 | 解释 
- | -
```SSR``` | Server Side Render，服务端页面渲染，本文档泛指React在Node服务端渲染。
```Isomorphic```  | 同构模式，指同一套代码在服务端和客户端都可以构建，本文档泛指一套React客户端代码可以在浏览器和服务端共用，达到渲染效果一致。
